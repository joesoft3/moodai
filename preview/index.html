Verdict: fine-tune first, build later if ever. Ghana should not train a frontier LLM from scratch — at ~$20M+ per competitive run and scarce H100-class capacity, the ROI is poor vs fine-tuning Llama 3 / DeepSeek on Twi, Ga, Ewe & Fante corpora (~$50–150k). Phase 1: national fine-tune + RAG over government/legal datasets. Phase 2: distillation for edge devices. Phase 3 (optional, 2028+): a small-from-scratch language-focused model once talent + compute pipelines mature…
Recommendation: fine-tune. Training from scratch would consume Ghana's entire AI-budget for marginal differentiation. Fine-tuning open weights delivers 90% of the value at ~1% of the cost; invest the difference in data infrastructure and university programs…
Skip sovereign training for now. The binding constraints are compute access, researcher retention and clean Ghanaian-language data — all better solved via fine-tuning + a national data trust. Revisit from-scratch training when regional compute cooperatives exist…
