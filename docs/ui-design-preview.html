Breakpoint tiers (tailwind.config.ts):
  • < 380 px (tiny phones) — composer buttons and paddings shrink; bottom-tab labels drop at <340 px leaving clean icons; nothing overflows.
  • < 768 px (phone) — top header with hamburger → full slide-over drawer; fixed bottom tab bar; env(safe-area-inset-*) padding; 100dvh layout avoids mobile URL-bar jumps.
  • 768–1024 px (tablet) — same drawer pattern (wider, 320 px); bottom bar disappears; roomier message column and image grids.
  • 1024–1536 px (laptop/desktop) — persistent 288–320 px sidebar with history + navigation; content centers in max-width columns.
  • 1536–1920 px (large monitors) — sidebar widens to 384 px; content widens; image grid 5-up.
  • > 1920 px (ultrawide) — content stays centered and capped.
  • Short / landscape screens (<560 px height) — vertical spacing compacts and the voice orb scales down automatically.

The live version also includes: a Voice Studio (pulsing orb state machine: idle → listening → thinking → speaking), an Image Lab (2/3/4-column gallery + lightbox), and a Settings page with the personal memory manager. Installable as a PWA via manifest.webmanifest.
